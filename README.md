# 🍷 Wine Quality Prediction & Analytics Dashboard

## 📌 Project Overview

This project focuses on analyzing and predicting wine quality using chemical properties from the Wine Quality dataset. The project combines **SQL** for data analysis, **Power BI** for dashboard visualization, and **Machine Learning concepts** for predictive analytics.

The main goal is to identify the key chemical factors that influence wine quality and present meaningful business insights through interactive dashboards. 📊

---

# 🎯 Objective

The objectives of this project are:

✅ Analyze wine quality using physicochemical attributes
✅ Perform data cleaning and preprocessing
✅ Use SQL for analytical calculations and insights
✅ Create an interactive Power BI dashboard
✅ Identify relationships between alcohol, acidity, and wine quality
✅ Generate KPIs using DAX measures
✅ Support data-driven quality assessment
